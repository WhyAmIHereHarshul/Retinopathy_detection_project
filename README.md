## Classification Model
# A simple CNN Model to classify between levels of Diabetic Retinopathy. 
Scans of retina are the inputs and the model classifies between the following categories:
1. No DR
2. Mild DR
3. Moderate DR
4. Severe DR
5. Proliferative DR
# Dataset taken from kaggle, aptos 2019 competition
Dataset link- https://www.kaggle.com/datasets/mariaherrerot/aptos2019
